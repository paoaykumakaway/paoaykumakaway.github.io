Project Timeline

Projected timeline for implementation of Paoay Kumakaway Application.

  • October 2016

    Project Kickoff

    This milestone is characterized by the contract agreement between stakeholders and developers on the implementation and scope details and Paoay Kumakaway Application content.

  • Nombember 2016

    GO LIVE: Initial Application Deployment

    This milestone is characterized by the release of the first working application, as has been defined in the contract agreement. The content of the first release includes the major modules of the application like Attractions, Foods, Accommodations and Events, as well as both the Android and Web applications for its administrators.

  • December 2016

    Development of iOS Application

    Development of iOS Application This milestone is characterized by the implementation of the iOS application development and release of initial modules.

  • January - March 2017

    Implementation of other features in mobile and web applications

    This milestone is characterized by the implementation of the remaining features in both mobile and web applications, as agreed upon in the contract agreement.

  • April - December 2017

    Support and Maintenance

    This milestone is characterized by the support and mintenance of the applications, as well as documentations and minor feature updates.
